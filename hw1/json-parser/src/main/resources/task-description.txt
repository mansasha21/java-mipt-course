Задание #1

Написать программу, принимающую на вход имя файла, содержащего Json с информацией о клиенте,
считать его и распарсить в экземпляр класса клиента соответствующего типа.
Тип клиента может отличаться. Набор полей класса для каждого типа клиента может отличаться.


Содержание файла:

name - наименование
inn - инн
Industry – принадлежность к отрасли

Типы клиентов:
INDIVIDUAL – физическое лицо,
LEGAL_ENTITY – юридическое лицо,
HOLDING – холдинг

LEGAL_ENTITY – юридическое лицо :

Пример json
{
   "name": "ООО Матрешка",
   "inn": 13242352,
   "clientType": "LEGAL_ENTITY"
}

INDIVIDUAL – физическое лицо :

Пример json
{
   "name": "Ivan",
   "surname": "Ivanov",
   "number": 111111
   "clientType": "INDIVIDUAL"
}

HOLDING – холдинг :

Пример json
{
   "name": "CHOCO",
   "zip": 111111,
   "regDate": "2021-01-01",
   "clientType": "HOLDING"
}


Для каждого типа создать отдельный класс.

Задачу решить двумя способами:
1) С помощью конструкции switch-case или if создать объекта класс в зависимости от типа.
2) Второй способ – тип клиента выделить в enum ClientType и использовать абстрактный метод createClient(…)

Написать unit тесты к решению. Junit подключить через gradle. Писать все в методе main не стоит. Вся логика должна быть в отдельных классах.
Парсинг Json-строки делаем руками. Стараемся переиспользовать код и избегать его дублирования.
Делаем отдельные классы, выделяем интерфейсы, используем enum - качество кода будет учитываться при проверке задания.
