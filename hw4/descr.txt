ДЗ по класслоудерам. Дедлайн до воскресенья следующей недели включительно.

Написать кастомный класслоудер, который умеет скачивать файлы по сети и дешифровывать массив байт.
В кастомном класслоудере необходимо переопределить метод findClass. Массив url для поиска классов можно передать в контруктор.
В файлах зашифрован сам .class файл и полное имя класса.
В первых n байтах содержится полное имя класса (вместе с пакетом), далее идет содержимое .class файла.

Для задания подготовлены два файла-примера

https://www.googleapis.com/drive/v3/files/1wG0bcva7AcA2v2TUEADYJFWCgSyL7KzN?alt=media&key=AIzaSyCnyt2lgtvTEVvITi-mD7v0s49OaxLcEog

Классы в указанных файлах реализуют интерфейс Runnable. В первом файле имя класса ru.sbt.java.tasks.Secret, во втором ru.sbt.java.tasks.VeryStrangeSecret.
Алгоритм шифрования такой: byte newByte = (byte) (oldByte + 1). После того как загрузите классы,
надо создать экзмпляр каждого класса и дернуть метод интерфейса, на консоль будет выведен секрет.
Загружать класс надо методом loadClass.
